# Buki Booking App

Prueba técnica full-stack para Buki Global.

## Estructura del Proyecto

- `backend/`: Node.js + Express (API REST)
- `frontend/`: Angular + Bootstrap (Web App)
- `mobile_flutter/`: Flutter (Opcional - Mobile App)
- `database/`: Scripts de base de datos MySQL

## Próximos pasos
El proyecto se irá desarrollando módulo por módulo según lo establecido en los requerimientos.
